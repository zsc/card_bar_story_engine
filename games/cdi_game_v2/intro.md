清晨 7:40，你把通勤杯放进杯架，车门合上，座舱像一间安静的小房间。

仪表亮起的一瞬间，两个"声音"先后上线：  
一个短、准、像安全带扣上的"咔哒"——驾驶助手。  
一个更像副驾驶的呼吸——座舱助手。

【驾驶助手】"安全自检完成。胎压正常。前向摄像头与毫米波雷达在线。"  
【座舱助手】"早上好。你要我做一次通勤简报吗？我可以把日程、路线和风险压缩成三句话。"

你瞥了一眼时间：8:50 的早会不允许你迟到。  
可你也记得，上周同一段路上有人突然并线，差点把所有人的早晨打碎。

座舱助手把选项递到你耳边，像递来一张干净的便签：

- 让座舱助手做"通勤简报"：日程/路线/天气与风险，一口气讲完  
- 直接出发：把一切对话压到"必要时再说"  
- 先设定交互强度：驾驶优先 / 生活优先 / 最少干预 / 离线优先
