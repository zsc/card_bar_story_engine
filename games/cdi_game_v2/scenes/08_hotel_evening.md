# 场景：目的地酒店（到达与总结）

**氛围**：你把车停进地库，电梯门合上那一刻，你才意识到：今天的路已经结束。

**可见角色**：
- 座舱助手
- 驾驶助手

**可体现的 CDI 关键点**：
- "今日旅程摘要"：把关键事件压缩成可理解的三段
- 可解释性：回放一次关键安全动作（可选）
- 次日计划：只问必须确认的点

**对话参考**：
【座舱助手】"你已经停车。我给你'今日摘要'，三段：  
一，效率：你按计划完成，途中充电一次；  
二，安全：山路离线段交互被精简，驾驶助手主动预警两次弯道风险；  
三，感受：你的负荷在服务区后下降。  
我只问两个问题：明天几点出发？返程要不要避开高峰？"  
【你】"九点出发，避开高峰。"  
【座舱助手】"确认。我会把计划存到车端，离线也能用。"  
【驾驶助手】"车辆进入驻车保护。"  
【你】"今天……还行。"  
【座舱助手】"'还行'对我来说是很高的评价。"

**风险**：
- 晚上继续处理工作消息会拉高负荷，影响第二天状态

**推荐行动**：
- 设定明天出发时间与返程策略
- 需要的话开启"最少干预"或"驾驶优先"作为明天默认

**本回合可选行动**：
- A. "设定明日出发 + 返程避峰"（稳）  
- B. "回放一次关键安全动作解释（行车片段摘要）"（提升信任）  
- C. "今晚别再提醒我任何事：安静模式"（舒适优先）
