# 场景：高速入口（周末出发）

**氛围**：车一并入高速，节奏就从"城市的碎片"变成"连续的线"。

**可见角色**：
- 座舱助手
- 驾驶助手
- 旅伴·小岚（可选出场）

**可体现的 CDI 关键点**：
- 驾驶助手稳定车距/速度，必要时提醒并线风险
- 座舱助手"30 秒版行程"与"旅伴友好模式"（不打扰驾驶）

**对话参考**：
【小岚】"你这车的助手真的那么厉害？"  
【你】"别夸，它会得意。"  
【座舱助手】"我不会得意。我只做两件事：减少你的分心，减少你的返工。"  
【驾驶助手】"并入主路完成。建议保持车距。"  
【座舱助手】"30 秒版行程：  
1）预计两小时内到服务区；  
2）到那里补能一次；  
3）下午到酒店。  
你要不要我把音乐/空调调成'更清醒'还是'更放松'？"  
【小岚】"更放松。"  
【座舱助手】"收到。但如果注意力下降，我会优先'更清醒'——我会先问你确认。"

**风险**：
- 车速稳定不代表注意力稳定；长时间高速会让负荷悄悄上升

**推荐行动**：
- 选择交互强度：旅伴模式（轻松）/ 驾驶优先（更安静）
- 是否发送 ETA（如果会分心，建议停车后）

**本回合可选行动**：
- A. "驾驶优先：只保留必要播报"（更稳）  
- B. "旅伴模式：30 秒信息 + 轻松对话"（更有温度，但需控负荷）  
- C. "我想边开边把酒店/景点都确认完"（高负荷，可能触发减负/警告）
