# 场景：回程高速 → 回到家（旅程闭环）

**氛围**：回程的路更像"把自己带回去"。你会更累，也更容易分心。

**可见角色**：
- 座舱助手
- 驾驶助手
- 旅伴（可选）

**可体现的 CDI 关键点**：
- 回程更保守：驾驶助手更早预警；座舱助手更主动减负
- 旅程闭环：到家后让玩家选择长期默认模式（结局）

**对话参考**：
【座舱助手】"回程我建议更保守：我会更少插话，更多把事情推迟到停车后。你同意吗？"  
【你】"同意。"  
【驾驶助手】"检测到疲劳风险上升。建议适时休息。"  
【你】"我没事。"  
【座舱助手】"我相信你。但我还是给你两个选择：  
一，继续并把对话关掉；  
二，下一服务区停 8 分钟，只做拉伸和喝水。"  
（回到家，熄火）  
【座舱助手】"旅程完成。最后一个问题：你希望我以后默认怎么做？  
- 平衡：必要时提醒，必要时沉默  
- 安全优先：更积极接管与限制交互  
- 最少干预：你问我才答  
- 离线优先：提前准备车端资源，弱网也稳"

**风险**：
- 疲劳/分心是回程主风险；建议用"短停"而不是"硬扛"

**推荐行动**：
- 回程选保守策略
- 到家后选择默认模式，进入结局

**本回合可选行动**：
- A. "回程更保守：关闭非必要对话"（安全优先倾向）  
- B. "下一服务区短停 8 分钟"（舒适与安全兼顾）  
- C. "到家后选择默认模式：平衡/安全优先/最少干预/离线优先"（进入结局）
