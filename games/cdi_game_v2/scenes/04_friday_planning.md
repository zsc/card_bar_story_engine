# 场景：家中车库（周末规划）

**氛围**：周五的车库比工作日更像"生活的入口"。你还没出发，但已经想离开城市半天。

**可见角色**：
- 座舱助手
- 驾驶助手（存在感更低，只做安全自检）

**可体现的 CDI 关键点**：
- 座舱助手用本地库给行程草案：路线、停靠、节奏
- 只在必要时请求确认：酒店/离线包/充电停靠/出发时间
- 强调 Edge：提前下载离线资源

**对话参考**：
【座舱助手】"你说想周末出去走走。我先给一个'不折腾'版本：  
- 上午出发，上高速；  
- 中途一次充电停靠；  
- 下午到酒店；  
- 第二天返程时避开拥堵时段。  
我需要你确认三个点：出发时间、是否走风景路、是否下载离线地图包。"  
【你】"你怎么不直接替我选？"  
【座舱助手】"因为这三个点会改变你的体验：  
风景路更好看，但弯多；离线包会占存储，但山路更稳；出发时间影响拥堵与疲劳。"  
【驾驶助手】"电池健康状态正常。制动系统自检通过。"  
【座舱助手】"要不要把旅伴小岚的上车点也加进计划？我可以只保留'必要提醒'，不在车动时展开解释。"

**风险**：
- 规划太细会增加负荷；但准备不足会在离线/续航时变得更焦虑

**推荐行动**：
- 选路线风格（快/稳/风景）
- 决定是否下载离线地图包
- 是否预置充电停靠

**本回合可选行动**：
- A. "快速路线 + 预置充电 + 下载离线包"（可靠性最高）  
- B. "风景路线 + 下载离线包"（体验优先，弯道更多）  
- C. "我不想被安排：最少干预，只要到时候提醒我就行"（自主但可能临时压力大）
